# fynth
fynth is a MIDI synthesizer for pipewire


# building
make


# usage
./fynth -f /usr/share/sounds/sf2/FluidR3_GM.sf2 -N


# why
fynth was a "can I do it"-challenge for myself in 2016.
August 15, 2021 I added pipewire support in it.
